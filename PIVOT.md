# Pivot Worksheet

## 1. Domain

**ชื่อธุรกิจ:** CookieCloudyDay

**ความเกี่ยวข้องกับนักศึกษา:** ธุรกิจที่อยากทำเอง / ธุรกิจที่สนใจทำในอนาคต

CookieCloudyDay เป็นร้านคุกกี้โฮมเมดที่ขายคุกกี้หลายรสชาติ เช่น คุกกี้ช็อกโกแลตชิพ คุกกี้เนยสด คุกกี้ช็อกโกแลตลาวา คุกกี้มัทฉะไวท์ช็อก และคุกกี้แมคคาเดเมียไวท์ช็อก โดยเน้นภาพลักษณ์ร้านขนมอบที่อบอุ่น เข้าถึงง่าย และเหมาะกับลูกค้าที่ต้องการซื้อกินเองหรือซื้อเป็นของฝาก

ธุรกิจนี้อยู่ในหมวดอาหาร/เครื่องดื่ม เพราะสินค้าเป็นขนมอบโฮมเมด และมีข้อมูลร้านที่ลูกค้ามักถามซ้ำ เช่น เมนู ราคา เวลาเปิด การจัดส่ง และช่องทางสั่งซื้อ

## 2. ลูกค้า + ปัญหา

**ลูกค้าหลักคือใคร:**  
ลูกค้าหลักคือ นักเรียน นักศึกษา คนทำงาน และลูกค้าที่ชอบขนมโฮมเมด รวมถึงคนที่ต้องการซื้อคุกกี้เป็นของฝาก

**3 ปัญหาที่ระบบจะแก้:**

1. ลูกค้าถามคำถามซ้ำ ๆ เช่น ร้านเปิดกี่โมง มีเมนูอะไรบ้าง มีบริการจัดส่งไหม และสั่งซื้อได้ทางไหน
2. ลูกค้าเลือกเมนูไม่ถูก เพราะมีคุกกี้หลายรสชาติและแต่ละคนมีความชอบต่างกัน เช่น ไม่ชอบหวานมาก หรืออยากซื้อเป็นของฝาก
3. เจ้าของร้านต้องเสียเวลาตอบ DM หรือข้อความเดิม ๆ หลายครั้ง ทำให้ตอบลูกค้าได้ช้าในช่วงที่มีออเดอร์เยอะ

## 3. Differentiation

**สิ่งที่ระบบนี้ทำได้ที่ MilkLab template ทำไม่ได้:**

ระบบของ CookieCloudyDay ไม่ได้เป็นเพียง chatbot สำหรับร้านเครื่องดื่มตามตัวอย่าง MilkLab แต่ปรับ domain ให้เป็นร้านคุกกี้โฮมเมดโดยเฉพาะ มี knowledge base ที่เก็บข้อมูลเมนูคุกกี้ ราคา จุดเด่นของแต่ละรส เวลาเปิดร้าน การจัดส่ง และคำถามที่ลูกค้าของร้านขนมมักถามจริง

จุดแตกต่างคือ chatbot สามารถช่วยแนะนำเมนูตามความต้องการของลูกค้าได้ เช่น เมนูขายดี เมนูที่ไม่หวานมาก เมนูที่เหมาะเป็นของฝาก และช่องทางการสั่งซื้อ โดยตอบจากข้อมูลร้าน CookieCloudyDay โดยตรง ไม่ใช่ตอบจากข้อมูลทั่วไปหรือจาก template ของ MilkLab

## Pivot Summary

จากเดิมที่เรียนด้วยตัวอย่าง MilkLab ในคอร์ส ฉัน pivot มาเป็น CookieCloudyDay ซึ่งเป็นธุรกิจร้านคุกกี้โฮมเมดในหมวดอาหาร/เครื่องดื่ม การ pivot นี้ไม่ได้เปลี่ยนแค่ชื่อร้าน แต่เปลี่ยนสินค้า กลุ่มลูกค้า pain point และข้อมูลใน knowledge base ให้เข้ากับธุรกิจใหม่

ระบบ AI ที่พัฒนาต่อจาก Session 1–3 จึงกลายเป็นผู้ช่วย AI สำหรับร้าน CookieCloudyDay ที่สามารถช่วยสร้าง caption บันทึกยอดขาย ส่งรายงาน และตอบคำถามลูกค้าผ่าน RAG chatbot ได้
